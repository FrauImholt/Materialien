Merksatz — Matrizenbegriff
Eine Matrix ist eine rechteckige Anordnung von Zahlen in Zeilen und Spalten. Die Dimension einer Matrix mit m Zeilen und n Spalten wird als m×n angegeben. Das Element in Zeile i, Spalte j heißt aij.

Merksatz — Mehrstufige Verflechtung
Wenn es mehrere Produktionsstufen gibt (Stufe 1 → Stufe 2 → Endprodukt), berechnet man schrittweise:
x₂ = B · d   →   x₁ = A · x₂   oder direkt:   x₁ = A · B · d
Die Matrizenmultiplikation ist assoziativ: (A·B)·d = A·(B·d) — beide Wege liefern dasselbe Ergebnis.
